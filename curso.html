IIND4417 Minería de Datos

Desarrolla habilidades para identificar patrones y extraer conocimiento valioso de conjuntos de datos masivos. Aprenderás a aplicar algoritmos y técnicas avanzadas para resolver problemas complejos del mundo real.

Información del Curso

Aprendizaje

Aprenderás a identificar problemas y seleccionar los métodos analíticos adecuados para realizar análisis estadísticos pertinentes en grandes conjuntos de datos.

Desarrollo de Proyectos

Desarrollarás proyectos de minería de datos, obteniendo relaciones entre variables e identificando patrones de comportamiento en sistemas asociados.

Manejo de Datos Masivos

Identificarás las dificultades que se presentan con el uso de bases de datos masivas y aprenderás a diferenciar la información requerida para distintos tipos de análisis.
